LearnLens AI — Dashboard Components

New components added under `src/components` and mock data under `src/utils/mockData.js`.

Dependencies to install in the Frontend folder:

```bash
cd Frontend
npm install recharts framer-motion prop-types lucide-react
```

Then start the dev server:

```bash
npm run dev
```

Notes:
- Components are React functional components and use Tailwind utilities defined in `index.css`.
- Use the mock data for local preview; API integration points remain in the existing pages.
